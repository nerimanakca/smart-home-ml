# smart-home-ml
Machine learning model for smart home device state classification using Random Forest, SVM and KNN algorithms.
# Smart Home Device State Classification

This project implements machine learning models to classify smart home device states into three categories:

- Normal
- Warning
- Critical

The goal is to analyze device behavior and predict system states using supervised learning algorithms.

## Algorithms Used
- Random Forest
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## Dataset
IEEE Smart Home Dataset

## Project Description
The dataset was preprocessed using data scaling and categorical encoding techniques. Multiple machine learning models were trained and evaluated to determine the most accurate classifier for device state prediction.
